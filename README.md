## ✨ Features

- 📊 Interactive CLI Dashboard  
- 📢 Mass Announcement System  
- 🏗️ Bulk Channel Creator (List Input)  
- 🧹 Channel Management (Create / Delete / List)  
- 👥 Member Moderation (Kick / Ban)  
- 🎭 Role Management System  
- 🏷️ Bulk Role Creator  
- 🧹 Message Cleaner  
- 📌 Mention Broadcaster  
- 📨 Embed Sender (Link → JSON → Send)  
- 📊 Server Info & Stats  
- 🔔 Bot Status Monitoring  
- 📜 Audit Log Viewer  
- 🚪 Safe Exit System

## 📦 Download

📁 Download the `.rar` file from the repository and extract it.

Inside:
- `Syembuk Server Manager.exe`

---

## 🚀 How to Use

1. Run the `.exe` file  
2. Enter your **Bot Token**  
3. Enter your **Server ID**  
4. Access dashboard  
5. Select features using numbers  

---

## ⚠️ Important

- Works within Discord API limits  
- Designed for controlled and responsible usage  
- Use only on servers you own or manage
